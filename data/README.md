# 🎂 ניהול תוכן Balicake — מדריך ללקוחה

ברוכה הבאה! המסמך הזה מסביר איך לעדכן את האתר שלך בעצמך, **בלי צורך בידע טכני**.

---

## ✏️ עדכון מוצרים, מחירים וסדנאות

כל המוצרים בקטלוג נשמרים בקובץ אחד: **[`data/products.json`](products.json)**

### איך לערוך?

1. כנסי ל-GitHub: https://github.com/yarinbrosh/balicake
2. לחצי על התיקייה **`data`** ואז על **`products.json`**
3. לחצי על האייקון של העיפרון ✏️ (פינה ימין-עליונה) — "Edit this file"
4. תערכי ישירות בדפדפן
5. גללי לתחתית → **"Commit changes"** → אישור
6. תוך **30 שניות** האתר מתעדכן אוטומטית!

### מבנה של מוצר (העתיקי-הדביקי כדי להוסיף חדש):

```json
{
  "id": "my-new-product",
  "category": "birthdays",
  "badge": "יום הולדת",
  "name": "שם המוצר",
  "price": "החל מ־₪320 · לפי משקל",
  "image": "https://קישור-לתמונה.jpg",
  "alt": "תיאור התמונה"
}
```

### הסבר על השדות:

| שדה | מה זה? |
|---|---|
| `id` | מזהה ייחודי (אנגלית-מקפים בלבד) |
| `category` | אחת מ: `birthdays`, `bento`, `bachelorette`, `course`, `3d`, `desserts` |
| `badge` | תווית שמופיעה מעל הכותרת (למשל "תלת ממד") |
| `name` | שם המוצר בעברית |
| `price` | המחיר — הקובץ מזהה אוטומטית את הסימן `₪` ומדגיש |
| `image` | קישור לתמונה (URL מלא, לא קובץ מקומי) |
| `alt` | תיאור התמונה לקוראי מסך ולנגישות |

### הקטגוריות הקיימות:

- `birthdays` — ימי הולדת ועיצוב עוגות
- `bento` — סדנאות בנטו
- `bachelorette` — סדנאות מסיבת רווקות
- `course` — קורסי אפייה
- `3d` — עוגות תלת מימד
- `desserts` — קינוחים במלאי

### איך לסדר את סדר הופעת המוצרים?

המוצרים מופיעים בקטלוג **לפי הסדר ב-JSON**. גזרי-הדביקי כדי לסדר.

---

## 📞 שינוי מספר הוואטסאפ של הזמנות

בקובץ `data/products.json` בתחתית:

```json
"_settings": {
  "whatsapp_phone": "972500000000",
  "whatsapp_message_template": "היי Balicake! אני רוצה להזמין: {name} ({price}). מה זמין?"
}
```

- שני את `whatsapp_phone` — בלי `+`, בלי `0` בהתחלה. **לדוגמה**: `972501234567`
- שני את `whatsapp_message_template` כדי לשנות את ההודעה האוטומטית. אל תיגעי ב־`{name}` ו־`{price}` — אלה החלפים אוטומטיים.

## 📸 איך מעלים תמונה חדשה?

**אופציה 1 — שירות חיצוני (הכי מומלץ):**
- עלי ל-https://imgur.com או https://cloudinary.com (חינם)
- העלי את התמונה
- העתיקי את ה-URL והדביקי ב-`image`

**אופציה 2 — ישר ל-GitHub:**
1. בריפו → תיקייה `images/` → "Add file" → "Upload files"
2. בחרי תמונה ושמרי
3. בקובץ `products.json` כתבי `"image": "images/שם-הקובץ.jpg"`

## 🆘 משהו השתבש?

אם האתר נשבר אחרי עדכון:

2. מצאי את ה-commit האחרון שעבד
3. לחצי "Revert" — האתר חוזר תוך 30 שניות
